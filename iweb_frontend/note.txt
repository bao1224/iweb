二、经典面试题：如何让内容在容器中竖直居中
方法1：父容器声明 line-height:内容高度
    局限：只适用于内容是一行文字
方法2：声明内容的margin-top
    局限：要求容器和内容的高度都是固定且已知
方法3：父容器display: table-cell;verical-align: middle
    局限：容器高度会固定，且兼容性不好
方法4：内容绝对定位，距离容器顶部50%，再向上微调上边距为负的自身高度的一半
    局限：内容的高度必需是一个已知且固定值
	.content {
		position:absolute;
		top:50%;
		margin-top:-xx%;
	}
方法5：CSS3提供的弹性布局
    局限：IE10及以下的浏览器不支持
	.box {
		display: flex;            /*容器变为弹性盒子*/
		align-items: center;      /*容器中的子元素对齐方式*/
	}
	.container {
		flex: 1;           /*子元素的尺寸占比*/
	}
